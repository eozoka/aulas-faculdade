# Repositório de Aulas da Faculdade

Este repositório foi criado para armazenar conteúdos, exercícios e projetos desenvolvidos durante as aulas da faculdade.

## Objetivo

Organizar e centralizar todo o material de estudo, facilitando revisões futuras e acompanhamento da evolução ao longo do curso.

## Tecnologias Utilizadas

* Python
* Java
* HTML
* React.js
* SQL / Banco de Dados
* Oracle Database
* Git e GitHub

## Conteúdo

* Exercícios práticos
* Trabalhos acadêmicos
* Projetos simples
* Testes e experimentos

## Observações

* Arquivos sensíveis como credenciais e bancos de dados não são versionados (ver `.gitignore`)
* Este repositório tem fins educacionais

## Autor

Desenvolvido durante a graduação.
